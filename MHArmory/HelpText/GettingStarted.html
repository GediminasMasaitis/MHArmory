Getting started To get started is very easy:
  1. Select some skills
    • On the main window, click the button Change skills (top left of the window)
    • In the skills selector window, check some skills
    • Close the skill selector window
  2. Start the search
    • On the main window, click the button Search (bottom left of the window)

Now you can do more advanced things, such as:
  • Selecting some weapon slots to increase the amount of jewels you can use.
    You can do this by changing the slots on the top toolbar.
  • Check weapons that match the selected weapon slots.
    You can do this by clicking the button with a Great Sword icon beside slots.
  • Override some decorations, to tell Armory about the jewels you have and you don't have.
    You can do this by changing the decorations override on the top toolbar.
    See Decorations override category in this help.
  • Override some equipment, to tell Armory about the equipment you have and you don't have.
    You can do this by changing the equipment override on the top toolbar.
    See Equipment override category in this help.
  • Sort armor set search results
    You can do this by clicking the Sort results... button on the top toolbar (top right).
    See the Sorting category in this help.
  • Load and save skill loadouts
    You can do this by clicking the Skill loadouts menu, then submenus.
  • Tweak equipments chosen for search results
    You can do this by clicking the Advanced search... button at the bottom left of the main window.
