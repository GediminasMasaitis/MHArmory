Text-based skill selection In the skills selection window, it is possible to use the search bar to quickly toggle skills selection.
When text is typed in the search bar, only matching results remain. For example if attack is typed, all skills containing attack in their name, their description, the description of the abilities it grants or the name of its related jewel(s), will remain in view.

It is also possible to append a number at the end of a search in order to express the will to match a given level for a given skill, for example attack 6. This will apply filter to keep all skills matching attack and that have a level or 6, meaning only Attack Boost should remain. Then pressing the Enter key will toggle the selection of the Attack Boost level 6 ability.
If speed 3 is typed, only Recovery Speed, Speed Eating and Speed Sharpening should remain. Now, pressing the Enter key will toggle selection of the level 3 of all the filtered skills.
